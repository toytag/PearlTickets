# ![logo](logo.png)


这个mod允许末影珍珠实体选择性地加载即将通过的区块，这样珍珠炮打出的珍珠就不会因为进入未加载区块而丢失。在1.14+中可以替代地狱门加载链使用。

该mod相比于@gnembon/carpet-extra mod的enderPearlChunkLoading功能有显著的性能提升
